# MidJourney Prompt Library

A quick index of reusable MidJourney prompt batches and where their images live.

## Prompt Batches

### Elf Dark Fantasy Prompt Pack 01

- **Asset folder:** `assets/images/npcs/core/elf/female/elf-dark-fantasy-pack-01/`
- **Descriptor note:** `assets/images/npcs/core/elf/female/elf-dark-fantasy-pack-01/index.md`
- **Exact prompt:**
  - `female elf, full body, fully visible from head to toe, centered single character, sharp features, dramatic clothing, shadowed mood, dark fantasy illustration, moody painted textures, grim elegance, gothic fantasy art, dramatic silhouette, subtle eerie atmosphere.`

## Usage Pattern

For each new prompt style, create one dedicated folder + `index.md` descriptor so prompt-matched images stay grouped and searchable.

- Example pattern:
  - `assets/images/npcs/<raceGroup>/<race>/<sexPresentation>/<prompt-pack-slug>/`
