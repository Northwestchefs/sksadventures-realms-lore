# AGENTS Instructions for realms-lore

## Repository Purpose
- This repository is a **Forgotten Realms lore vault** designed for both **Obsidian** and **Quartz** publishing.
- Treat all content as player-safe, campaign-usable reference material unless a page explicitly states otherwise.

## Authoring Philosophy
- Prefer **scaffold-first pages** over encyclopedic lore dumps.
- Keep entries concise, structured, and easy to expand over time.
- Favor reusable structure and cross-linking over maximum lore density.
- Keep pages useful for both quick browsing and future deep-build passes.

## Markdown and Linking Standards
- Use plain Markdown with clean heading hierarchy (`#`, `##`, `###`).
- Use **Obsidian-style wiki links** throughout (for example: `[[nations-realms/cormyr|Cormyr]]`).
- Keep formatting Quartz-friendly; avoid complex or fragile rendering patterns.

## Canon and Timeline Caution
- Treat canon-sensitive facts carefully.
- Do **not** overcommit to disputed rulers, exact dates, edition-specific contradictions, or narrow timeline claims unless those claims are already established in-repo.
- When canon varies by era or source, prefer broad, campaign-usable phrasing.
- It is acceptable for pages to remain intentionally light where certainty is low.

## Repository Structure Rules
- Keep the existing folder architecture intact.
- Do not delete or rename established files without a strong, explicit reason.
- When expanding a topic, prefer creating useful related stubs over overloading a single page.
- Maintain strong hub/index pages that support navigation and backlink discovery.

## Scope Expectations for Future Work
- Continue building durable content layers across:
  - nations-realms
  - settlements
  - regions
  - faiths
  - factions
  - people
  - noble-houses
  - landmarks
- Prioritize consistency with existing page schema and style anchors already present in the repository.
