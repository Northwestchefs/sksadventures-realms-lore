# Expanded Species NPC Images

Use this folder for official expanded playable 5e species/races outside the core list (for example: aarakocra, bugbear, changeling, firbolg, genasi, goblin, kenku, kobold, tabaxi, warforged, yuan-ti).

Pattern:

- `assets/images/npcs/expanded/<race>/<sexPresentation>/<npc-slug>/`
